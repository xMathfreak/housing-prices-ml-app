# housing-prices-ml-app
This project uses machine learning to predict house prices based on various features, such as the size of the house, number of bathrooms and quality of the kitchen. The model is trained using the [Kaggle House Prices dataset](https://github.com/djeada/kaggle-house-prices).

## Technologies Used
+ Python
+ Pandas
+ Scikit-learn
+ Streamlit
+ Matplotlib
+ Seaborn
+ Pickle

## How to use
1. Clone the repository
2. Install the required dependencies: `pip install -r requirements.txt`
3. Run the Streamlit app: `streamlit run app.py`
4. Enter the house features to get a price prediction
